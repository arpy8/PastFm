## Description
PastFm is a simple github readme card to display your listening activity.

## Setting up the Python API with `uv`
1. Setup the project using:
   ```sh
   uv venv
   ```
2. Install dependencies:
   ```sh
   uv pip install -r requirements.txt
   ```
3. Run the API:
   ```sh
   python api.py
   ```
